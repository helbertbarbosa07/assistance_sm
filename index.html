Tecnologias utilizadas:

LGPD - Lei Geral de Proteção de Dados

Política de Privacidade e Proteção de Dados

Coleta de Dados

Este sistema coleta apenas dados necessários para seu funcionamento:

  • Informações de usuário (nome, username, senha criptografada)
  • Informações sobre cursos cadastrados
  • Feedbacks e avaliações (opcional)
  • Histórico de conversas com a IA (opcional)

Armazenamento de Dados

Todos os dados são armazenados localmente no seu navegador usando IndexedDB. Nenhuma informação é enviada para servidores externos, exceto quando utiliza a API de IA (OpenRouter).

Direitos do Usuário

De acordo com a LGPD, você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Excluir seus dados a qualquer momento
  • Revogar consentimento

Segurança

Implementamos medidas de segurança para proteger seus dados:

  • Armazenamento local seguro
  • Senhas criptografadas
  • Acesso restrito por autenticação
  • Limpeza de cache opcional
